## 🌟 Overview

This project is a **fully interactive Power BI dashboard** built to analyze vehicle sales data across multiple dimensions like:

* 💰 Revenue
* 📦 Quantity Sold
* 🌍 Country Performance
* 🚘 Product Line
* 📊 Deal Size

## 🔥 Key Insights

* 🚘 **Classic Cars** generate highest revenue (~2.97M)
* 🌍 **USA dominates sales (~34%)**
* 📊 **Medium Deal Size is most profitable**
* 📉 Revenue peaks in **2014**, then declines
* 🚢 Ships & Trains contribute least
